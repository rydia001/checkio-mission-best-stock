現在の株価(Stock price)が与えられます。どの株が最も高いか見つけてください。

入力: 市場のIDコードがキーで、その株価が値である辞書(dict)。

出力: 市場のIDコードを表す文字列。

例:

{% if interpreter.slug == "js-node" %} 
bestStock({
    'CAC': 10.0,
    'ATX': 390.2,
    'WIG': 1.2
}) == 'ATX'
bestStock({
    'CAC': 91.1,
    'ATX': 1.01,
    'TASI': 120.9
}) == 'TASI'
{% else %} 
best_stock({
    'CAC': 10.0,
    'ATX': 390.2,
    'WIG': 1.2
}) == 'ATX'
best_stock({
    'CAC': 91.1,
    'ATX': 1.01,
    'TASI': 120.9
}) == 'TASI'
{% endif %}

条件: 同じ価格は複数存在しない。
