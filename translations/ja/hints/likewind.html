どうやって解けばいいかまったくわかりません。
最初に、一番高い株価の名前として変数に0をセットします。次に最初の株価と0にセットされた変数の値を比較し、同様に次の株価とも比較していきます。 もし新しい株価がより高ければ、その値が一番高い株価に入れられ、その名前を記録します。すべての株価を見たとき、名前の変数に入っている株が答えです。
どうすれば全ての株価を順番に比較できますか？
for文を使います。
コードの例をください。（訳注：答えが含まれます！）
今回の例ではこのようになります： 
def best_stock(data):    
    max_price = 0
    answer = ''
    for s in data:
        if data[s] > max_price:
            max_price = data[s]
            answer = s
    return answer
